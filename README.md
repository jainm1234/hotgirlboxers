# Hot Girl Boxers

A one-page trendy website for Hot Girl Boxers — underwear that’s 100% cotton, breathable, and built to feel good and look good. Inspired by [Good Girl Snacks](https://goodgirlsnacks.com).

## What’s included

- **Hero** — Marquee strip + headline and CTA
- **Shop** — 3 product placeholder cards (Classic Fit, Mid-Rise, Long Leg)
- **Why Ours** — Product details: 100% cotton, no pee pee flab, butt seam, perfect length
- **Our Story** — About section (started to avoid UTIs and yeast infections)
- **Newsletter** — Sign-up block
- **Footer** — Links and copyright

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
# Python 3
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Tech

- Plain HTML5, CSS3, minimal JS (year in footer)
- Semantic markup, ARIA where helpful, responsive layout
- Fonts: Playfair Display (display), Outfit (body) via Google Fonts
